# Codex Documentation Structure

## Files

| File | Description |
|------|-------------|
| `using-codex/cli/overview.md` | Introduction to Codex CLI, setup instructions, and overview of key features |
| `using-codex/cli/features.md` | Detailed guide to CLI features including interactive mode, resuming conversations, models and reasoning |
| `using-codex/cli/command-line-options.md` | Command-line options and flags reference |
| `using-codex/cli/slash-commands.md` | Complete guide to built-in slash commands like /model, /approvals, /status, /review |
| `configuration/config-file/basic-configuration.md` | Basic Codex configuration file settings |
| `configuration/config-file/sample-configuration.md` | Sample configuration file examples |
| `configuration/rules.md` | Rules system for controlling which commands Codex can run outside the sandbox |
| `configuration/agents-md.md` | Custom instructions with AGENTS.md files for global and project-specific guidance |
| `configuration/custom-prompts.md` | Creating custom prompts that behave like slash commands |
| `configuration/mcp.md` | Model Context Protocol (MCP) integration for third-party tools and context |
| `configuration/skills/overview.md` | Agent skills system for extending Codex with task-specific capabilities |
| `configuration/skills/create-skill.md` | Step-by-step guide to creating custom skills |
